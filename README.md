# SSM_art
本系统是在SSM框架以及SpringBoot的基础上进行开发的，采用了安全框架Spring Security，功能模块包含了人事管理、信息管理、公告管理、客服管理以及订单管理。
其中支付功能使用的是沙箱模拟支付宝支付！！
编译器采用的是IDEA2021<br>
